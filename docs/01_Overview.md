# 01 — Описание продукта

## Цель проекта

Дать руководителю продукта в Лемана Про инструмент быстрой и убедительной оценки бизнес-эффекта любой фичи или инициативы. Каждый калькулятор закрывает один типовой вопрос: «Сколько денег / NPS / конверсии мы получим, если сделаем X?»

## Целевая аудитория

- **Primary:** Дмитрий Татьков (РП в Лемана Про) — для подготовки питчей CPO Сукновалова и аргументации в Demand-management комитете
- **Secondary:** другие продакты ЛП — могут пользоваться готовыми калькуляторами и шаблонами для своих кейсов

## Принципы

1. **Один калькулятор = один вопрос.** Не пытаемся сделать «универсальную» формулу, делаем точечные инструменты под конкретные кейсы.
2. **Метрики — из единого источника.** Baseline тянем из Google Sheets, чтобы не было расхождений между калькуляторами.
3. **Визуализация важнее формул.** Каждый расчёт сопровождается графиком/таблицей с анимациями — для CPO-питчей это критично.
4. **Сценарное моделирование — в каждом калькуляторе.** Любой калькулятор позволяет менять входные метрики и мгновенно видеть результат. Пользователь сохраняет несколько вариантов (Сценарий A / B / C...) и сравнивает их на дашборде бок-о-бок. Это не дополнительная фича — это обязательная часть каждого плагина.
5. **История сценариев → детализированный дашборд.** Все сохранённые сценарии отображаются на общем дашборде с возможностью наложения, сравнения и drill-down в детали каждого.
6. **Бренд ЛП.** Палитра, типографика, логотип — соответствуют гайдлайнам.

## Scope MVP

**В MVP входит:**
- Каталог калькуляторов (минимум 1, далее расширяем)
- Дашборд с анимированными метриками (Recharts/ECharts + Framer Motion)
- Авторизация (Firebase Auth, доступ только для приглашённых email)
- Источник baseline-метрик — Google Sheets через Service Account
- Сохранение сценариев в Firestore
- Деплой на Vercel с привязанным доменом

**В MVP НЕ входит:**
- Multi-tenant (только команда ЛП)
- Экспорт в PDF / pptx (потом)
- Совместное редактирование сценариев в реальном времени
- Мобильная версия (адаптив — да, нативные приложения — нет)

## Метрики успеха

| Метрика | Target |
|---|---|
| Калькуляторов в каталоге к концу Q3 2026 | ≥ 8 |
| Время на расчёт нового сценария | < 3 мин |
| Использований при подготовке питчей CPO | ≥ 5 кейсов |
| Расхождение с финальными цифрами в защите | < 10% |
