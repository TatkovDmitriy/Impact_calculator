## Новоселы: распределение по AOV-сегментам (2026)

Сегментация платящих клиентов-Новоселов по среднему чеку (AOV) за 2026 год — кухни, ванные, хранение, межкомнатные двери.

**Метрика:** AOV клиента = суммарный GMV / количество оплаченных проектов.

**Источник данных:** ячейка 23 (`sql_aov_segments`) ноутбука «Потенциал Новоселов 07.05.2026.ipynb».

**Примечание:** сегменты AOV взяты из ноутбука (5 диапазонов: до 50к / 50–100к / 100–150к / 150–200к / 200к+). В исходном задании указаны 3 диапазона (до 100к / 100–300к / 300к+) — расхождение согласовано с PM.

**График** показывает распределение числа клиентов по сегментам.

**Таблица** содержит по каждому сегменту: количество клиентов, долю в трафике, GMV, долю GMV, маржинальность, среднее число оплат на клиента.
