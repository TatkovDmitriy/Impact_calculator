## Новоселы: частота покупок vs База (дек 2025 – апр 2026)

Сравнение среднего количества оплаченных проектов на одного клиента между сегментом Новоселов и базой клиентов.

**Метрика:** среднее число оплаченных проектов на уникального клиента (`paid_deals / unique_clients`) по каждому месяцу.

**Источник данных:** ячейка 15 (`sql_freq_total`) ноутбука «Потенциал Новоселов 07.05.2026.ipynb».

**Примечание:** данные охватывают всю клиентскую базу (все типы сделок). Метрика «средняя частота оплат» используется как прокси для повторных покупок — ячейка с прямой долей клиентов с 2+ покупками в разрезе Новоселов в ноутбуке отсутствует.

**График** показывает динамику частоты оплат по месяцам для каждого сегмента.

**Таблица** содержит помесячную детализацию с количеством клиентов и средней частотой для обоих сегментов.
